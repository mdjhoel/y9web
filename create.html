Provided code - Apply it for your data and go beyond!

  • HTML basic structure of a site
  • HTML intermediate topics: canvas, button with onclick="myfunction()" and onchange="otherfunction()", select pulldown
  • Basic CSS definition for HTML tag ids, mycanvas { background-color: red; }
  • Basic JS: document.getElementById("mycanvas"), using canvas tag functions
  • Intermediate JS: function myfunction { }, use variables, string concatination ("con" + "catenation"), loops and if statements, innerHTML to populate select pulldown, or sequential search (for loop with if statement inside)

Rubric

3-4

i. constructs a detailed and logical plan, which describes the efficient use of time and resources, sufficient for peers to be able to follow to create the solution

ii. demonstrates satisfactory technical skills (apply most/all provided code for your data) with guidance

iii. follows the plan to create the solution, which functions as intended and is presented appropriately

iv. outlines changes made to the chosen design when making the solution.

5-6

i. constructs a detailed and logical plan, which describes the efficient use of time and resources, sufficient for peers to be able to follow to create the solution

ii. demonstrates competent technical skills (apply provided code for your data with some novel updates) with some guidance

iii. follows the plan to create the solution, which functions as intended and is presented appropriately

iv. describes changes made to the chosen design when making the solution.

7-8

i. constructs a detailed and logical plan, which describes the efficient use of time and resources, sufficient for peers to be able to follow to create the solution

ii. demonstrates excellent technical skills (goes beyond provided code) with minimal guidance; a self starter and self sufficient

iii. follows the plan to create the solution, which functions as intended and is presented appropriately

iv. fully justifies changes made to the chosen design when making the solution.
